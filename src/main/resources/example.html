Web‑Beispiele für die API

Interaktive Beispiele zur Demonstration von HTTP‑Methoden und Statuscodes. Jede Aktion zeigt eine Vorhersage und das tatsächliche Ergebnis.

Mehr Demos: /status/all Startseite

GET /user?id=...

Gib eine ID ein und teste per GET (ergebnis inline oder neuer Tab).

Noch keine Anfrage.

POST /user

Erstelle Benutzer per HTML‑Form oder JSON (inline Ergebnis).

Noch keine Anfrage.

PUT / PATCH / DELETE

Buttons senden PUT, PATCH und DELETE per Fetch; Resultat erscheint unten.

Keine Aktion ausgeführt.
